## What's inside?
This boilerplate is a mixture of:

- [Initializr](http://www.initializr.com/)
- [Bootstrap v4-alpha](https://github.com/twbs/bootstrap)

## Dependencies

- [Node](https://nodejs.org/en/download/)
- npm install -g grunt-cli
- npm install
